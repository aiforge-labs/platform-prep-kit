Help & Support

Everything you need to run Platform Prep Kit locally, get the most out of it, and contribute back. All data stays on your machine in ~/.prep/.

Running locally

Get the web UI up in under a minute. Requires Python 3.10+.

  1. Install 
    pip install platform-prep-kit[ui]
# or, from a clone:
pip install -e ".[ui]"
  2. Start the server 
    prep serve
    Opens this dashboard at http://127.0.0.1:8080.
  3. Run a quiz from the CLI (optional) 
    prep quiz --tag mitre-atlas

Trouble? Check /health returns {"status":"ok"}, and that port 8080 isn't already in use (lsof -iTCP:8080 -sTCP:LISTEN).

Using the app

  • Today — your current day's study card. Mark done when finished.
  • Study Plan — full multi-week plan, filterable by week and track.
  • Quizzes — practice questions tagged by topic (MITRE ATLAS, NIST AI RMF, OWASP LLM Top 10, etc.).
  • Knowledge — deep-dive packs you can check off section by section.
  • Portfolio — role-specific project templates with rubrics, to show on your CV/LinkedIn.
  • Settings — change role template, reminder schedule, integrations.

Contributing

Found a bug, want to add a knowledge pack, or have a new role template to ship? Contributions are welcome.

  1. Fork the repo on GitHub
  2. Create a feature branch (git checkout -b feat/your-idea)
  3. Run the test suite: make test
  4. Open a pull request — small, focused PRs merge fastest

Read CONTRIBUTING.md for code style, commit-message conventions, and how knowledge packs / project templates are structured.

Good first contributions

  • Add a knowledge pack for a topic you know well (Markdown file in knowledge_packs/)
  • Add quiz questions tagged by topic (quiz_banks/*.yml)
  • Add a role template (project_templates/*.yml) — RAG eval, security ML, MLOps, etc.
  • Improve docs or fix a typo

Get support

  • Report a bug or request a feature (GitHub Issues)
  • Discuss ideas & share your study plans (GitHub Discussions)
  • Report a security issue privately
